# VueJS Directives and Routing Training

## Overview
Mastering VueJS directives and client-side routing with Vue Router.

## Training Content

### 1. Built-in Directives
- Conditional directives (v-if, v-show)
- List rendering (v-for)
- Event handling (v-on)
- Two-way binding (v-model)

### 2. Custom Directives
- Directive registration
- Directive hooks
- Directive parameters
- Directive best practices

### 3. Vue Router Fundamentals
- Router installation and setup
- Route configuration
- Dynamic route matching
- Nested routes

### 4. Advanced Routing
- Route guards
- Route meta fields
- Lazy loading routes
- Scroll behavior

### 5. Navigation Patterns
- Programmatic navigation
- Named routes
- Route props
- Navigation best practices

## Hands-on Exercises
1. Create a custom directive for special UI behavior
2. Build a complex navigation system
3. Implement route guards for authentication
4. Develop a lazy-loaded application
5. Create a custom router transition system

## Assessment
- Coding challenge: Build a complete routed application
- Debugging exercise: Fix routing and directive issues
- Quiz on directive patterns and routing best practices
