# Spring Boot Maven Web Application Training

## Overview
Creating and configuring Spring Boot web applications using Maven build system.

## Training Content

### 1. Maven Fundamentals
- Maven project structure
- POM.xml configuration
- Dependency management
- Build lifecycle

### 2. Spring Boot with Maven
- Spring Boot starter dependencies
- Parent POM inheritance
- Plugin configuration
- Multi-module projects

### 3. Web Application Basics
- Spring MVC architecture
- Controller annotation
- Request mapping
- View resolution

### 4. RESTful Web Services
- REST principles
- @RestController annotation
- Request/Response handling
- HTTP methods and status codes

### 5. Web Application Configuration
- Static resources handling
- Template engines (Thymeleaf, Freemarker)
- Webjars integration
- CORS configuration

## Hands-on Exercises
1. Create a Maven-based Spring Boot web application
2. Build a simple REST API with CRUD operations
3. Implement a web application with Thymeleaf templates
4. Configure CORS for a frontend application
5. Create a multi-module web application

## Assessment
- Coding challenge: Build a complete web application
- Debugging exercise: Fix Maven and web configuration issues
- Quiz on Spring MVC and REST principles
