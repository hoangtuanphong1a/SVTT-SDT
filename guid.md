1. Nội dung đào tạo Java Core
Giai đoạn Java Core tập trung trang bị kiến thức nền tảng về ngôn ngữ Java, bao gồm:
•	Cài đặt môi trường, sử dụng Notepad để viết chương trình Java đầu tiên và dùng CMD để compile, run.
•	Các khái niệm cơ bản: Object, Class, Method, Variable; cú pháp và Access Modifier; comments trong Java.
•	Object, Class, Constructor, Variable; kiểu dữ liệu nguyên thủy và kiểu dữ liệu tham chiếu; escape sequences.
•	Các loại toán tử: Arithmetic, Relational và các toán tử khác.
•	Vòng lặp (Loop) và các class kiểu số như Integer, Double.
•	Character, String và Array.
•	Date, DateFormat và Method.
•	File I/O.
•	Exception Handling.
•	Kế thừa (Inheritance) và đa hình (Polymorphism – Overriding, Overloading).
•	Abstract Class và Interface.
•	Collection Framework.


2. Nội dung đào tạo Spring Boot
Giai đoạn Spring Boot giúp sinh viên làm quen với framework backend hiện đại:
•	Tìm hiểu tổng quan về Spring Boot.
•	Tạo và sử dụng Maven Web Application.
•	Xây dựng một ứng dụng CRUD.
•	Tìm hiểu các loại Web Service: SOAP và RESTful.
•	Chuyển ứng dụng CRUD sang mô hình microservices, giao tiếp qua RESTful API.
•	Hoàn thiện ứng dụng thành một dự án hoàn chỉnh, tương tác ít nhất 3 bảng có mối quan hệ.

4. Nội dung đào tạo Docker
Sinh viên được giới thiệu và thực hành các nội dung sau:
•	Docker Compose.
•	Docker Swarm.

5. Nội dung đào tạo VueJS
Giai đoạn VueJS tập trung vào xây dựng frontend động:
•	Tổng quan VueJS và thiết lập môi trường.
•	Vue instances và template.
•	Component trong Vue.
•	Computed properties và watch property.
•	Data binding và event handling.
•	Rendering.
•	Transition và animation.
•	Directives và routing.
•	Mixins và Render Functions.
•	Lifecycle Hooks.
•	Chuyển đổi frontend sang ứng dụng VueJS hoàn chỉnh.
